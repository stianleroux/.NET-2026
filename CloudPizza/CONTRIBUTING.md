# Contributing to CloudPizza

Thank you for your interest in contributing! This is a demo application designed for conference talks.

## How You Can Help

### Use It for Your Talks
- Fork this repo
- Customize for your presentation
- Add your own features
- Share your improvements!

### Improve the Demo
- Fix bugs
- Add comments/documentation
- Improve code examples
- Add tests

### Share Your Experience
- Open issues for questions
- Share how you used it in your talk
- Suggest improvements

## Code Standards

This project demonstrates best practices, so contributions should follow:
- ✅ SOLID principles
- ✅ Clean Architecture
- ✅ Modern .NET 10 features
- ✅ Comprehensive comments explaining "why"
- ✅ Self-documenting code

## Pull Request Process

1. Fork the repo
2. Create a feature branch (`git checkout -b feature/amazing-improvement`)
3. Make your changes
4. Ensure code builds and runs
5. Update documentation if needed
6. Commit with descriptive messages
7. Push and create a Pull Request

## Questions?

Open an issue or reach out to the maintainers.
